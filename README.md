# 🌿 منصة الداعية المفكر

### منصة المسابقات الدعوية والمعرفية الإلكترونية

منصة عربية متكاملة لإدارة المسابقات والاختبارات الإلكترونية والشهادات،
مبنية بأحدث التقنيات وبواجهة عربية أنيقة تدعم الوضع الليلي والنهاري.

`React 19` · `TypeScript` · `Vite` · `Tailwind CSS v4` · `Zustand` · `Framer Motion`

</div>

---

## 📖 نظرة عامة

**الداعية المفكر** منصة تعليمية تنافسية تتيح للمتسابقين خوض اختبارات
متعددة ضمن مسابقات مختلفة، والحصول على شهادات معتمدة عند النجاح،
مع لوحة تحكم شاملة للمسؤول تتيح إدارة كل شيء دون الحاجة لأي برمجة.

---

## ✨ أبرز المميزات

### 👤 للمتسابق
- **شاشة ترحيب** أنيقة بفيديو تعريفي وزر «ابدأ الرحلة» (وضع ملء الشاشة).
- **تسجيل دخول مرن:** بالاسم أو البريد أو رقم الهاتف، أو الدخول كزائر، أو عبر جوجل.
- **حساب كامل:** اسم رباعي، بريد، واتساب، نوع، صورة شخصية، واختيار أكثر من مسابقة.
- **اختبارات ذكية:**
  - سحب أسئلة عشوائي مختلف في كل محاولة.
  - ترتيب عشوائي للاختيارات.
  - عدّاد زمني أنيق وصور للأسئلة بتحميل فوري.
  - زر «تخطّي» للأسئلة وإرجاعها في النهاية.
- **شاشة تحليل النتيجة:** الدرجة، النسبة، التقدير، رسالة تحفيزية، ومراجعة الأخطاء.
- **شهادة معتمدة** قابلة للطباعة و التحويل إلى PDF.
- **لوحة الشرف، الإحصائيات، المذكرات، الفيديوهات، الشروحات، التعليمات، التعليقات.**
- **مساعد ذكي** يجيب عن كل ما يخص المنصة، متاح من شاشة الترحيب لكل الزوار.

### 👑 لمسؤول المنصة (لوحة التحكم)
| القسم | التحكم |
|------|--------|
| الإعدادات العامة | الاسم، الشعار، الترحيب، شريط الإعلان، العلامة المائية، أيقونات التواصل |
| ألوان المنصة | تحكم كامل في ألوان الوضع النهاري والليلي |
| شكل العرض | عدد الكروت، شكل الاختيارات، الأقسام الظاهرة، الوضع الافتراضي |
| المتسابقون | إضافة/تعديل/حذف، حظر/إيقاف، اشتراك، محاولات، بصمة الجهاز، استيراد Excel |
| المسابقات | إضافة وتعديل المسابقات |
| الاختبارات | عدد الأسئلة، الوقت، نسبة النجاح، للمشتركين، يظهر لأكثر من مسابقة |
| بنوك الأسئلة | إضافة يدوية + استيراد من Excel / Google Sheets + صور وشرح |
| المذكرات والفيديوهات | إضافة يدوية أو **جلب تلقائي للبيانات من الرابط**، لكل المسابقات أو محددة |
| الشروحات | مجلدات ومقالات منسّقة أو روابط خارجية |
| الاشتراك | السعر، المميزات، طريقة الدفع، رقم واتساب |
| المساعد الذكي | تفعيله ومراجعة كل المحادثات وتعديل ردوده |
| إعدادات SEO | عنوان ووصف وكلمات مفتاحية وصورة المشاركة |
| روابط الشيتات | إضافة روابط Google Sheets والدرايف |

## 🗂️ هيكل المشروع

```
src/
├── components/        # المكوّنات (الهيدر، الفوتر، المساعد، إلخ)
│   └── admin/         # مكوّنات لوحة التحكم
├── pages/             # صفحات المنصة
├── lib/
│   ├── types.ts       # تعريفات الأنواع
│   ├── store.ts       # حالة التطبيق (Zustand) + الحفظ المحلي
│   ├── defaults.ts    # البيانات الافتراضية
│   ├── utils.ts       # دوال مساعدة
│   ├── importer.ts    # استيراد CSV / Google Sheets
│   └── linkMeta.ts    # جلب بيانات الروابط تلقائياً
└── App.tsx            # التوجيه والإعداد العام
```
